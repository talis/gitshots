Talis Git Shots!

What the funk is this?

Take a picture and collect some stats every time you commit!

Installation Instructions:

  1. Download post-commit.py
  2. Install this as a post-commit hook in your repo, like this!
    3. This is how I do this:
    1. git config --global init.templatedir '~/.git_template'
    2. mkdir -p ~/.git_template/hooks
    3. echo '#!/bin/sh\n/usr/bin/env python2.7 /FULLPATH/TO/post-commit.py' >> ~/.git_template/hooks/post-commit
    4. chmod +x ~/.git_template/hooks/post-commit
  3. Run git init in a repo where you want to use the hook!
